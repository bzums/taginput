jquery.taginput Examples and API

Examples

Apply different ui-styles

You can apply different ui-styles on the widget using the style option. 

$('#taginput-jqueryui').({
	style: 'jquery'
});

$('#taginput-bootstrap').taginput({
	style: 'bootstrap'
});
